# better-content

Own-your-data, adapter-driven, framework-agnostic inline-edit CMS engine. MIT.

Edit content directly on your live pages, persist it to **your** database
through a small adapter seam — no hosted service, no lock-in. The engine is
framework-free; React is the first binding.

> **Status: pre-release.** Nothing published yet; the API is still forming.
> See [`docs/`](./docs/README.md) for the architecture and build plan.

Independent project — not affiliated with better-auth or the better-\* family;
it just shares the values: own your data, framework-agnostic, TypeScript-first,
MIT.
